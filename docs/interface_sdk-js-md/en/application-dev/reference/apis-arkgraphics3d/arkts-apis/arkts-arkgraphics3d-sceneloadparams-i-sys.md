**Since:** 26.0.0

## offset

```TypeScript
offset?: number
```

The offset of the start of the 3D model data in the resource Unit: byte, The value must be greater than or equal to 0. Default value: 0.

**Type:** number

**Default:** { 0 }

**Since:** 26.0.0

**Model restriction:** This API can be used only in the stage model.
