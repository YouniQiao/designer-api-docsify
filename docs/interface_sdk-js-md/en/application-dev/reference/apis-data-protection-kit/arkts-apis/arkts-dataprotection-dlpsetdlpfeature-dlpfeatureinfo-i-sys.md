## dlpFeatureStatus

```TypeScript
dlpFeatureStatus: DlpFeatureStatus
```

DLP status, which can be set to **NOT_ENABLED_FEATURE** or **ENABLED_FEATURE**.

**Type:** DlpFeatureStatus

**Since:** 26.0.0

**Model restriction:** This API can be used only in the stage model.
