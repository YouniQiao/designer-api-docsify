# TextVerticalAlign

Defines the vertical alignment mode of text. The default value is **BASELINE** (aligning along the baseline).

**Since:** 20

<!--Device-unnamed-declare enum TextVerticalAlign--><!--Device-unnamed-declare enum TextVerticalAlign-End-->

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## BASELINE

```TypeScript
BASELINE = 0
```

Aligns text along the baseline.

**Since:** 20

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 20.
