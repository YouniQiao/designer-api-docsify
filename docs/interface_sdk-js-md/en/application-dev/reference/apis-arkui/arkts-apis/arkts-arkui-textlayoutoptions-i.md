## constraintWidth

```TypeScript
constraintWidth?: LengthMetrics
```

Layout width of the measured text. If not set, the width is the maximum width occupied by a single-line layout.

**Type:** LengthMetrics

**Since:** 20

**Model restriction:** This API can be used only in the stage model.
