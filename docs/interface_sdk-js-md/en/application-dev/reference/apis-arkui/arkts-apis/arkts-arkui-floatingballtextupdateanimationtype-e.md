# FloatingBallTextUpdateAnimationType

Enumerates the animation types used when the floating ball text is updated.

## ANIMATION_NONE

```TypeScript
ANIMATION_NONE = 0
```

No animation.

**Since:** 26.0.0

**Model restriction:** This API can be used only in the stage model.
