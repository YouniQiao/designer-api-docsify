## uiExtensionHostWindowProxyRect

```TypeScript
uiExtensionHostWindowProxyRect: window.Rect
```

Position, width, and height of the **UIExtensionComponent**.

**Type:** window.Rect

**Since:** 11

**Model restriction:** This API can be used only in the stage model.
