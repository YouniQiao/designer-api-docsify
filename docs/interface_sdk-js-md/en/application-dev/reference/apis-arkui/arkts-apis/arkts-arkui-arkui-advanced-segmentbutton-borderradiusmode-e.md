# BorderRadiusMode

Enumerates the border radius modes for the **SegmentButton** component, which are used to control the border radius calculation method.

**Since:** 20

## DEFAULT

```TypeScript
DEFAULT = 0
```

Default mode, where the framework automatically calculates the border radius.

**Since:** 20

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 20.
