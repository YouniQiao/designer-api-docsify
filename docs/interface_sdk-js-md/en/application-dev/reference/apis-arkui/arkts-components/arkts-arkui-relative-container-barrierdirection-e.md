# BarrierDirection

Defines the direction of a barrier.

**Since:** 12

<!--Device-unnamed-declare enum BarrierDirection--><!--Device-unnamed-declare enum BarrierDirection-End-->

**System capability:** SystemCapability.Test.UiTest

## LEFT

```TypeScript
LEFT
```

The barrier is on the left side of all the referenced components specified by [referencedId](arkts-arkui-relative-container-barrierstyle-i.md).

**Since:** 12

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 12.
