# ComputedBarAttribute

Provides information about the position and length of the scrollbar.

**Since:** 10

<!--Device-unnamed-declare interface ComputedBarAttribute--><!--Device-unnamed-declare interface ComputedBarAttribute-End-->

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## totalLength

```TypeScript
totalLength: number
```

Total length of the grid content, in px.

**Type:** number

**Since:** 10

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 11.

## totalOffset

```TypeScript
totalOffset: number
```

Total offset of the grid content relative to the display area, in px.

**Type:** number

**Since:** 10

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 11.
