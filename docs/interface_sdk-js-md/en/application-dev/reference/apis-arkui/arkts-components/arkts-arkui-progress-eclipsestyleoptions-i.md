# EclipseStyleOptions

Options of the eclipse style. The eclipse style visualizes the progress in a way similar to the moon waxing from new to full.

Inherits from [CommonProgressStyleOptions](arkts-arkui-progress-commonprogressstyleoptions-i.md).

**Inheritance/Implementation:** EclipseStyleOptions extends [CommonProgressStyleOptions](arkts-arkui-progress-commonprogressstyleoptions-i.md)

**Since:** 10

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 11.
