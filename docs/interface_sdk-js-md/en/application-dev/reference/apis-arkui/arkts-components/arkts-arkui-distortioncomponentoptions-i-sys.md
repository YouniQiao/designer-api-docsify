## distortion

```TypeScript
distortion?: DistortionParam
```

Spatial distortion parameters. The spatial distortion effect is generated by specifying the position relationship of four corners and the barrel distortion degree of four edges.

**Type:** DistortionParam

**Since:** 26.0.0

**Model restriction:** This API can be used only in the stage model.
