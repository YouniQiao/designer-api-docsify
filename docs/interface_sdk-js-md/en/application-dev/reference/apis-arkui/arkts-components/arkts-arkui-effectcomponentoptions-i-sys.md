## effectLayer

```TypeScript
effectLayer?: EffectLayer
```

Rendering layer of the EffectComponent.

Default value: EffectLayer.NONE

**Type:** EffectLayer

**Since:** 20

**Model restriction:** This API can be used only in the stage model.
