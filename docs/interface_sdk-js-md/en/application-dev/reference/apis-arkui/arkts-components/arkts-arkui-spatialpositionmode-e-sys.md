# SpatialPositionMode (System API)

Spatial position mode. Indicates the coordinate system used by the corner positions.

**Since:** 26.0.0

<!--Device-unnamed-declare enum SpatialPositionMode--><!--Device-unnamed-declare enum SpatialPositionMode-End-->

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System API:** This is a system API.

## WORLD_XYZ

```TypeScript
WORLD_XYZ = 0
```

World XYZ coordinate mode. The X, Y, and Z components are all in world coordinates.

**Since:** 26.0.0

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 26.0.0.

## NDC_XY_WORLD_Z

```TypeScript
NDC_XY_WORLD_Z = 1
```

NDC XY and world Z coordinate mode. The X and Y components use NDC (Normalized Device Coordinates),and the Z component is in world coordinates.

**Since:** 26.0.0

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 26.0.0.
