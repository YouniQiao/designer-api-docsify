# MeasureResult

Provides the measurement result of the component. This API inherits from [SizeResult](arkts-arkui-common-sizeresult-i.md).

**Inheritance/Implementation:** MeasureResult extends [SizeResult](arkts-arkui-common-sizeresult-i.md)

**Since:** 10

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 11.
