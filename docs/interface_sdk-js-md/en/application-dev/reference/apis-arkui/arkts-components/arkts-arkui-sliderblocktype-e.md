# SliderBlockType

Enumerates the types of the slider in the block direction.

| Name | Value| Description |  
| ------- | -- | ---------------------- |  
| DEFAULT | 0 | Round slider. |  
| IMAGE | 1 | Slider with an image background. |  
| SHAPE | 2 | Slider in a custom shape.|

**Since:** 10
