# ParamType

```TypeScript
type ParamType = number | number | number | string | boolean | Array<string>
```

Enumerates the types of custom event parameter values.

**Since:** 12

**Atomic service API:** This API can be used in atomic services since API version 12.

<!--Device-hiAppEvent-type ParamType = int | long | double | string | boolean | Array<string>--><!--Device-hiAppEvent-type ParamType = int | long | double | string | boolean | Array<string>-End-->

**System capability:** SystemCapability.HiviewDFX.HiAppEvent

| Type | Description |
| --- | --- |
| int | Int. |
| long | Long. |
| double | Double. |
| string | String. |
| boolean | The value is true or false. |
| Array&lt;string&gt; | The value is an array of strings. |
