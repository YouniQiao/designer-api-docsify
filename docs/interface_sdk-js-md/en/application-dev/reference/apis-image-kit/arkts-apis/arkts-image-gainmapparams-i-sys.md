## isFullSizeGainmap

```TypeScript
isFullSizeGainmap: boolean
```

Indicates generating a full-size gainmap or a 1/2 downscaled gainmap.

**Type:** boolean

**Since:** 26.0.0

**Model restriction:** This API can be used only in the stage model.
