# AlternateIconInfo

Describes the app backup icon information.

**Since:** 26.0.0

## enabled

```TypeScript

```

Whether the backup icon is enabled.true: The current backup icon is enabled.false: The current backup icon is not enabled.

**Type:** boolean

**Since:** 26.0.0
