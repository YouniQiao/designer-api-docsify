# AutoFillPopupConfig (System API)

The module describes the size and position information of an auto-fill pop-up.

**Since:** 12

<!--Device-unnamed-export default interface AutoFillPopupConfig--><!--Device-unnamed-export default interface AutoFillPopupConfig-End-->

**System capability:** SystemCapability.Ability.AbilityRuntime.AbilityCore

**System API:** This is a system API.

## placement

```TypeScript
placement?: PopupPlacement
```

Position of the auto-fill pop-up. If this parameter is not set, the position is not updated.

**Type:** PopupPlacement

**Since:** 12

**Model restriction:** This API can be used only in the stage model.

<!--Device-AutoFillPopupConfig-placement?: PopupPlacement--><!--Device-AutoFillPopupConfig-placement?: PopupPlacement-End-->

**System capability:** SystemCapability.Ability.AbilityRuntime.AbilityCore

**System API:** This is a system API.

## popupSize

```TypeScript
popupSize?: PopupSize
```

Width and height of the auto-fill pop-up. If this parameter is not set, the width and height are not updated.

**Type:** PopupSize

**Since:** 12

**Model restriction:** This API can be used only in the stage model.
