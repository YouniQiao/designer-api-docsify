## MidiSysex

```TypeScript
MidiSysex = "TYPE_MIDI_SYSEX"
```

MIDI SYSEX resource.

Currently, only permission events can be reported. MIDI devices are not yet supported.

**Since:** 9

**Atomic service API:** This API can be used in atomic services since API version 11.
