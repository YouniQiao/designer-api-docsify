# NestedScrollOptionsExt

Implements a **NestedScrollOptionsExt** object to set up, down, left, and right nested scrolling options.

**Since:** 14

## scrollDown

```TypeScript
scrollDown?: NestedScrollMode
```

Default value: **NestedScrollMode.SELF_FIRST**.
