## BLE_SCAN_ENHANCE_MODE_BALANCED

```TypeScript
BLE_SCAN_ENHANCE_MODE_BALANCED = 0
```

Balanced scan performance and other bluetooth service performance.

**Since:** 26.0.0

**Model restriction:** This API can be used only in the stage model.


