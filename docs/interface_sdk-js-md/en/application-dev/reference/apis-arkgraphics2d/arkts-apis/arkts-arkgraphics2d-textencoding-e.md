# TextEncoding

Enumerates the text encoding types.

**Since:** 11

## TEXT_ENCODING_UTF8

```TypeScript
TEXT_ENCODING_UTF8 = 0
```

One byte is used to indicate UTF-8 or ASCII characters.

**Since:** 11

**Atomic service API:** This API can be used in atomic services since API version 22.

## TEXT_ENCODING_UTF16

