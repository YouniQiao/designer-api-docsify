# Constants

## DATA_ROAMING_STATUS

```TypeScript
const DATA_ROAMING_STATUS: string
```

Specifies whether data roaming is enabled.

<p>If the value is {@code true}, data roaming is enabled. If the value is {@code false},data roaming is disabled.

**Since:** 7

**Deprecated since:** 21
