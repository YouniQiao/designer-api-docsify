# EventPolicy

提供系统事件配置策略的定义，用于使用[configEventPolicy](arkts-performanceanalysis-configeventpolicy-f.md#configeventpolicy-1)设置事件配置策略。

**起始版本：** 22

**系统能力：** SystemCapability.HiviewDFX.HiAppEvent

## addressSanitizerPolicy

```TypeScript
addressSanitizerPolicy?: AddressSanitizerPolicy
```

地址越界事件配置策略。

**类型：** AddressSanitizerPolicy

**起始版本：** 24

**元服务API：** 从API版本24开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.HiviewDFX.HiAppEvent
