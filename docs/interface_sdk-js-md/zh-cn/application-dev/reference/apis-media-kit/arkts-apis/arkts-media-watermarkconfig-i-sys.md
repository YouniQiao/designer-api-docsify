# WatermarkConfig（系统接口）

设置AVRecorder的水印配置。水印位置从左上角开始计算。

**起始版本：** 13

**系统能力：** SystemCapability.Multimedia.Media.Core

**系统接口：** 此接口为系统接口。

## left

```TypeScript
left: number
```

水印到左侧像素行的偏移量。

**类型：** number

**起始版本：** 13

**系统能力：** SystemCapability.Multimedia.Media.Core

**系统接口：** 此接口为系统接口。

## top

```TypeScript
top: number
```

水印到顶部像素行的偏移量。

**类型：** number

**起始版本：** 13

**系统能力：** SystemCapability.Multimedia.Media.Core

**系统接口：** 此接口为系统接口。
