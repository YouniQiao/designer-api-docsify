# DistributedHardwareErrorCode（系统接口）

分布式硬件错误码的枚举。

**起始版本：** 11

**系统能力：** SystemCapability.DistributedHardware.DistributedHardwareFWK

**系统接口：** 此接口为系统接口。

## ERR_CODE_DISTRIBUTED_HARDWARE_NOT_STARTED

```TypeScript
ERR_CODE_DISTRIBUTED_HARDWARE_NOT_STARTED = 24200101
```

表示分布式硬件未启动。

**起始版本：** 11

**系统能力：** SystemCapability.DistributedHardware.DistributedHardwareFWK

**系统接口：** 此接口为系统接口。

ERR_CODE_DEVICE_NOT_CONNECTED = 24200102
