# ShareOptions

UDMF支持的设备内使用范围类型枚举。

**起始版本：** 12

**系统能力：** SystemCapability.DistributedDataManager.UDMF.Core

## IN_APP

```TypeScript
IN_APP = 0
```

表示允许在本设备同应用内使用。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.DistributedDataManager.UDMF.Core

## CROSS_APP

```TypeScript
CROSS_APP = 1
```

表示允许在本设备内跨应用使用。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.DistributedDataManager.UDMF.Core
