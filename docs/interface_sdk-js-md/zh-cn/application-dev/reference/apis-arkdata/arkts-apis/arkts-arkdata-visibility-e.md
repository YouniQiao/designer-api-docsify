# Visibility

表示数据的可见性等级枚举。

**起始版本：** 20

**系统能力：** SystemCapability.DistributedDataManager.UDMF.Core

## ALL

```TypeScript
ALL
```

可见性等级，所有应用可见。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.DistributedDataManager.UDMF.Core

## OWN_PROCESS

```TypeScript
OWN_PROCESS
```

可见性等级，仅数据提供者可见。

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.DistributedDataManager.UDMF.Core
