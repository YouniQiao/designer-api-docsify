# PlainText

[Text](arkts-arkdata-text-c.md)的子类，用于描述纯文本类数据。

**继承/实现关系：** PlainText extends [Text](arkts-arkdata-text-c.md)

**起始版本：** 10

**系统能力：** SystemCapability.DistributedDataManager.UDMF.Core

## abstract

```TypeScript
abstract?: string
```

纯文本摘要，非必填字段，默认值为空字符串。

**类型：** string

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.DistributedDataManager.UDMF.Core
