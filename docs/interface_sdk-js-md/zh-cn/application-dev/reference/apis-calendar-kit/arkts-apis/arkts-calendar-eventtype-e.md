# EventType

日程类型枚举。

**起始版本：** 10

**系统能力：** SystemCapability.Applications.CalendarData

## NORMAL

```TypeScript
NORMAL = 0
```

普通日程，例如会议，闹钟等日常提醒的日程。

**起始版本：** 10

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Applications.CalendarData

## IMPORTANT

```TypeScript
IMPORTANT = 1
```

重要日程，例如结婚纪念日等具有重要意义的日期，不推荐三方开发者使用，重要日程类型不支持一键服务跳转功能及无法自定义提醒时间。

**起始版本：** 10

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Applications.CalendarData
