# DpiFollowStrategy（系统接口）

表示不同类型的DpiFollowStrategy的枚举。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

## FOLLOW_HOST_DPI

```TypeScript
FOLLOW_HOST_DPI = 0
```

跟随宿主DPI。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

## FOLLOW_UI_EXTENSION_ABILITY_DPI

```TypeScript
FOLLOW_UI_EXTENSION_ABILITY_DPI = 1
```

跟随UIExtensionAbility。

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。
