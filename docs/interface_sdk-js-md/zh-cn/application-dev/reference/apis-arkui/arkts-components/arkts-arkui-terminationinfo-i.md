# TerminationInfo

表示嵌入式UI的提供方终止时的信息。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## code

```TypeScript
code: number
```

定义终止码。

**类型：** number

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## want

```TypeScript
want?: import('../api/@ohos.app.ability.Want').default
```

定义额外的终止信息。

**类型：** import('../api/@ohos.app.ability.Want').default

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
