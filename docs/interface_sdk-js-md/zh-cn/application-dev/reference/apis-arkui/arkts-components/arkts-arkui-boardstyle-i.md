## borderRadius

```TypeScript
borderRadius?: Length
```

背板的圆角半径（不支持百分比设置）。

默认值：8.0

单位：vp

取值范围：[0, +∞)。异常值时取默认值。

**类型：** Length

**默认值：** 8.0vp [since 11]

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API版本11开始，该接口支持在原子化服务API中使用。
