# TimeFormat

TimeFormat枚举用于定义时间选择器的格式。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## HOUR_MINUTE

```TypeScript
HOUR_MINUTE = 0
```

显示时、分。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## HOUR_MINUTE_SECOND

```TypeScript
HOUR_MINUTE_SECOND = 1
```

显示时、分、秒。

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
