# DatePickerComponent

DatePickerComponent组件用于选择日期（年月日）和时间（时分秒）。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## options

```TypeScript
options: DatePickerComponentOptions
```

定义日期时间选择器组件的选项。

**类型：** DatePickerComponentOptions

**起始版本：** 26.0.0

**装饰器类型：** @Prop

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
