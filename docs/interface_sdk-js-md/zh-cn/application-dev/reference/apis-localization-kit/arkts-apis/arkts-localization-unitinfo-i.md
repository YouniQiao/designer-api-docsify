# UnitInfo

度量衡单位信息。

**起始版本：** 8

**系统能力：** SystemCapability.Global.I18n

## measureSystem

```TypeScript
measureSystem: string
```

单位的度量体系，取值包括：'SI', 'US', 'UK'。

**类型：** string

**起始版本：** 8

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.I18n

## unit

```TypeScript
unit: string
```

单位的名称，如：'meter', 'inch', 'cup'等。

**类型：** string

**起始版本：** 8

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.I18n
