# StyledNumberFormatOptions

创建富文本显示的数字格式化对象时的可选配置项。

**起始版本：** 18

**系统能力：** SystemCapability.Global.I18n

## decimal

```TypeScript
decimal?: TextStyle
```

指定小数点的文本样式。默认值：StyledString默认的文本样式。

**类型：** TextStyle

**起始版本：** 18

**元服务API：** 从API版本18开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.I18n
