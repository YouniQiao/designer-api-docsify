# HolidayLocalName

节假日名称在不同语言下的翻译。

**起始版本：** 11

**系统能力：** SystemCapability.Global.I18n

## language

```TypeScript
language: string
```

语言，例如ar，en，tr。

**类型：** string

**起始版本：** 11

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.I18n

## name

```TypeScript
name: string
```

节假日的本地名称，例如Sacrifice Feast（宰牲节）的土耳其语名称为Kurban Bayrami。

**类型：** string

**起始版本：** 11

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.I18n
